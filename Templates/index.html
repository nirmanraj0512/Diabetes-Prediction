About This Tool

This diabetes risk assessment tool uses machine learning to predict the likelihood of diabetes based on key health indicators. The model was trained on clinical data using a K-Nearest Neighbors algorithm and achieves 78% accuracy.


Please note this tool is for informational purposes only and should not replace professional medical advice. Always consult a healthcare provider for medical concerns.
